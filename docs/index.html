pklib.ajax {Object}

{XMLHttpRequest} pklib.ajax.load ( {Object} config )

  • Opis
  • Przykład

Tworzy zapytanie do serwera. Zapytanie może być synchroniczne.

type {String}
Typ zapytania (get | post)
async {Boolean}
Flaga określająca status zapytania asynchronicznego
cache {Boolean}
Flaga określająca czy odpowiedź ma być cache'owana
url {String}
Adres zapytania
params {Object}
Parametry zapytania które mają być wysłane razem z zapytaniem
timeout {Number}
Maksymalna długość zwłoki na odpowiedź z serwera. Wartość w milisekundach
headers {Object}
Nagłówki zapytania
done {Function}
Funkcja która uruchomi się po wykonaniu zapytania
<script type="text/javascript">
pklib.ajax.load({
    cache: true,
    url: "http://example.org",
    params: {
        test: 1
    },
    done: function () {
        console.log("Yes!");
    }
});
</script>

pklib.array {Object}

{Boolean} pklib.array.isArray ( {Object} array )

  • Opis
  • Przykład

Sprawdzenie czy parametr jest instancja obiektu Array 

<script type="text/javascript">
pklib.array.isArray([1,2]); // true
pklib.array.isArray({a:1}); // false
pklib.array.isArray(null); // false
pklib.array.isArray(1); // false
pklib.array.isArray("A"); // false
</script>

{Boolean} pklib.array.inArray ( {any} param, {Object} array )

  • Opis
  • Przykład

Sprawdzanie czy parametr należy do tablicy 

<script type="text/javascript">
pklib.array.inArray(1, [1,2]); // true
pklib.array.inArray([1], [1,2]); // false
pklib.array.inArray(1, {a:1}); // false
pklib.array.inArray(1, null); // TypeError: array is null
pklib.array.inArray(1); // TypeError: array is undefined
pklib.array.inArray("A"); // false
</script>

pklib.loader {Object}

Obsługa loadera, który jest dodawanym do drzewa DOM elementem o określonych parametrach. Wymiary obrazka jak i link do obrazka jest łatwo konfigurowalny. Gdy tylko pokażemy loader za pomocą metody **pklib.loader.show** zostanie on z default'u wycentrowany. Kontener w którym chcemy umieścić (pre)loader jest łatwy do zmiany, gdyż nie zawsze chcemy aby loader znajdował się na środku całej strony, ale być może tylko w danym kontenerze na stronie.

pklib.message {Object}

Top layer, który jest ponad wszystkimi elementami w drzewie DOM. Pomocny gdy chcemy wyświetlić informację, wiadomość bądź komunikat użytkownikowi. Dzięki dobremu algorytmowi, warstwa której jasno zdefiniujemy wymiary zawsze pozostanie na środku strony. Obłsuga layera posiada "zamykanie", a tak na prawdę usuwanie z drzewa DOM elementu będącego top layerem, layera, tj. usunięcie z widoku warstwy reprezentującej komunikat dla użytkownika.
